# Chattomate

Backend server for Chattomate:
* [NodeJS v14.17.1](https://nodejs.org/en/)
* [Mongodb v4.4.3](https://www.mongodb.com/)


## Installation

Use the package manager [npm](https://www.npmjs.com/) to run.

```bash
npm install
npm run start
```
